FORECAST

A convenient extension for detailed Faceit statistics.

This project is a browser extension that collects and displays detailed statistics for Faceit matches and players.

Key features include:

  • Adds new levels
  • Shows player statistics in match history
  • Provides access to match tabs from history
  • Calculates win rates for teams and players in the match room

It displays detailed player stats in match history.

Shows:

  • Match score
  • HLTV rating 2.1 (~7%)
  • Kills and deaths
  • KD and KR ratios
  • Average Damage per Round (ADR)

It also adds quick access to view the match scoreboard directly from the history.

It adds extra rating levels for a more accurate reflection of player skill at higher ranks.

It expands the standard 10-level Faceit system, making the ranking more detailed.

It shows extended info about teams and individual players in the match room, calculating the number of games and win percentage on each map.

This helps better assess team strengths on specific maps before the game starts.

Match Analyze Limit
30

It lets you adjust the number of matches used for analysis within a match.

Has no effect if the 'Extended in-match info' setting is turned off.
